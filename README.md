# Knowledge Snapshot (Exam Management System)

Простая и удобная платформа для управления экзаменами, созданная на Django. Позволяет создавать экзамены, прикреплять к ним студентов и группы, загружать изображения и управлять видимостью событий.

## 🚀 Основные возможности

*   **Список экзаменов**: Просмотр всех опубликованных экзаменов в виде карточек.
*   **Поиск и фильтрация**: Умный поиск по названию экзамена, имени студента или названию группы.
*   **Управление экзаменами**:
    *   Создание и редактирование экзаменов.
    *   Загрузка обложки (изображения) для экзамена.
    *   Настройка даты и времени проведения.
    *   Управление видимостью (`Опубликован` / `Скрыт`).
*   **Управление участниками**:
    *   Прикрепление отдельных студентов к экзамену.
    *   Прикрепление целых **Групп** студентов (все участники группы автоматически добавляются к экзамену).
*   **Управление группами**:
    *   Создание и редактирование учебных групп.
    *   Массовое добавление студентов в группы.
*   **Административная панель**: Полный контроль над данными через стандартную админку Django.

## 🛠 Технологии

*   **Python 3.10+**
*   **Django 5+**
*   **Pillow** (для работы с изображениями)
*   **HTML5 / CSS3** (собственный стиль без использования тяжелых CSS-фреймворков)
*   **SQLite** (база данных по умолчанию)

## 📦 Установка и запуск

Следуйте инструкции ниже, чтобы развернуть проект локально.

### 1. Клонирование репозитория

```bash
git clone <URL_вашего_репозитория>
cd knowledgeSnapshot
```

### 2. Создание и активация виртуального окружения

**macOS / Linux:**
```bash
python3 -m venv venv
source venv/bin/activate
```

**Windows:**
```cmd
python -m venv venv
venv\Scripts\activate
```

### 3. Установка зависимостей

```bash
pip install -r requirments.txt
```
*(Если файла requirments.txt нет, установите основные пакеты вручную: `pip install django pillow`)*

### 4. Применение миграций

Создайте базу данных и таблицы:

```bash
python manage.py makemigrations
python manage.py migrate
```

### 5. Создание суперпользователя (Админа)

Чтобы зайти в админку, создайте пользователя:

```bash
python manage.py createsuperuser
```
*(Следуйте инструкциям в терминале, введите логин, email и пароль)*

> **Для быстрого старта:** В проекте может быть уже создан пользователь `admin` с паролем `admin` (если вы использовали скрипт инициализации).

### 6. Запуск сервера

```bash
python manage.py runserver
```

Проект будет доступен по адресу: [http://127.0.0.1:8000/](http://127.0.0.1:8000/)

## 🔗 Ссылки

*   **Главная страница (Список экзаменов):** `/exam/` или просто `/`
*   **Админка:** `/admin/`
*   **Добавление экзамена:** `/exam/add/`
*   **Список групп:** `/groups/`

## 📂 Структура проекта

*   `core/` — Основные настройки проекта (settings, urls).
*   `exams/` — Приложение с логикой экзаменов (models, views, forms).
    *   `templates/exams/` — HTML шаблоны.
    *   `static/exams/` — CSS стили.
*   `media/` — Папка для загруженных изображений.
*   `db.sqlite3` — Файл базы данных.
